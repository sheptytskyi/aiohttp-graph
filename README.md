# AioHTTP GraphQL Interview Task
<hr>

## Tech. Stack:
- Python
- AioHTTP
- SqlAlchemy
- Graphene
- PostgreSQL

## Usage:
1. Create venv
```python
python3 -m venv venv
```

2. Activate it
```python
source venv/bin/activate
```
3. Install requirements
```python
pip install -r requirements.txt
```
4. Startup server
```python
python app.py
```

5. Go to endpoint
```python
http://localhost:8080/graphql
```
